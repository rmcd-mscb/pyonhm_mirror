# pyONHM Mirror

This repository is a mirror of the pyONHM (Python Operational National Hydrologic Model) work from the USGS GitLab.

## Description

pyONHM is a Python package that provides a Command Line Interface for managing Docker images used to run the U.S. Geological Survey Operational National Hydrologic Model. The model is a CONUS-wide watershed model driven by gridMET climate forcings on a daily basis. Additionally, it supports:

- **Sub-seasonal forecasts**: Using the downscaled CFSv2 product of 48, 28-day ensembles delivered daily
- **Seasonal forecasts**: Using the downscaled NMME product of 6-month forecasts delivered monthly

## Source Repository

The original source code is maintained on the USGS GitLab. This GitHub repository serves as a public mirror for visibility and collaboration.
